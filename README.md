# 🚀 Dev-QC Workflow Kanban

Dashboard Kanban interaktif yang dirancang khusus untuk memantau alur kerja antara tim Developer dan tim QC secara real-time.

## ✨ Fitur Utama
- **Smart Timer**: Pantau durasi pengerjaan setiap task (Dev & QC).
- **Priority Labels**: Penanda visual P0 hingga P3 dengan warna kontras.
- **Team Assignment**: Penugasan anggota tim dengan avatar inisial.
- **Audit Trail**: Log otomatis setiap kali status task berubah.
- **Responsive Design**: Tampilan colorful dan modern yang nyaman di mata.

## 🛠️ Teknologi yang Digunakan
- **Framework**: [Next.js](https://nextjs.org/)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **Icons**: [Lucide React](https://lucide.dev/)
- **Deployment**: [Vercel](https://vercel.com/)
